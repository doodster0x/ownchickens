# OwnChickens.com

The beginner's guide to backyard laying hens.

## Structure

- `index.html` — homepage
- `CNAME` — custom domain for GitHub Pages

## Deploying

This site is hosted on GitHub Pages. Push to `main` and changes go live automatically.

## Adding pages

Add new `.html` files to the root for each guide section:
- `breeds.html`
- `housing.html`
- `feeding.html`
- `care.html`
- `laws.html`
- `sourcing.html`
